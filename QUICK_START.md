# Quick Start Guide

Follow these steps to get your portfolio live on GitHub Pages:

## Step 1: Upload Files to Your Repository

1. Download all files from this folder
2. Go to your GitHub repository: `JAWAHAR-Eng.github.io`
3. Upload the following structure:

```
JAWAHAR-Eng.github.io/
├── index.html
├── README.md
├── .gitignore
└── assets/
    ├── css/
    │   └── style.css
    ├── js/
    │   └── script.js
    ├── images/
    └── resume/
        └── Jawahar_Bharanitharan_Resume.pdf  (YOU NEED TO ADD THIS)
```

## Step 2: Add Your Resume

**IMPORTANT**: Add your resume PDF to `assets/resume/` with the exact name:
`Jawahar_Bharanitharan_Resume.pdf`

## Step 3: Enable GitHub Pages

1. Go to your repository Settings
2. Click on "Pages" in the left sidebar
3. Under "Source":
   - Branch: `main` (or `master`)
   - Folder: `/ (root)`
4. Click "Save"

## Step 4: Wait for Deployment

- GitHub will build and deploy your site
- This usually takes 1-5 minutes
- You'll see a green checkmark when it's ready

## Step 5: Visit Your Site

Your portfolio will be live at:
**https://JAWAHAR-Eng.github.io**

## Optional: Add Custom Domain

If you have a custom domain:
1. In GitHub Pages settings, add your custom domain
2. Configure DNS records at your domain provider
3. Enable "Enforce HTTPS"
